### Objetivo do software
O sistema **OficinaPro** tem como objetivo facilitar o gerenciamento de oficinas mecânicas, permitindo o cadastro de clientes, veículos, serviços e ordens de serviço, tornando o processo mais ágil e organizado.

### Funcionalidades do sistema
- Cadastro de clientes  
- Cadastro de veículos  
- Registro de serviços realizados  
- Emissão de ordens de serviço  
- Controle de status das ordens  
- Consulta e histórico de serviços
